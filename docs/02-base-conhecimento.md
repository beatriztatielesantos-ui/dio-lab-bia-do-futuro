# 📊 Base de Conhecimento

## 📌 Visão Geral

A base de conhecimento simula um ambiente corporativo de aprendizagem e desenvolvimento.

---

## 📁 Fontes de Dados

### 1. perfil_investidor.json
Representa o perfil do colaborador:
- Nome
- Cargo
- Objetivo de carreira

---

### 2. Estrutura de Cursos (simulada)

O agente considera cursos como:
- Comunicação Efetiva
- Fundamentos de Liderança
- Inteligência Emocional

---

## 🧠 Estratégia de Uso

O agente utiliza uma abordagem simples de RAG:

1. Lê o perfil do colaborador
2. Identifica o objetivo (ex: liderança)
3. Associa cursos relevantes
4. Gera recomendações personalizadas

---

## 💡 Possível Evolução

- Integração com LMS (ex: Imagine)
- Uso de dados reais de trilhas
- Recomendações baseadas em desempenho
